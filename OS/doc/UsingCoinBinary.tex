\title{Using the CoinEasy Software}
\vskip 2in
\author{Horand Gassmann, Jun Ma,  Kipp Martin}
\maketitle

\begin{abstract}
This is the User's Manual for the CoinEasy project which is designed to help new users of COIN-OR get up and running. 
It is intended for users of COIN-OR who are interested in solving optimization problems but {\it do not} wish to compile source code. 
 In particular, we show how the user can send optimization problems to a COIN-OR server and get results back.  The COIN-OR server, \url{webdss.ise.ufl.edu},   is 2x Intel(R) Xeon(TM) CPU 3.06GHz 512MiB L2 1024MiB L3, 2GiB DRAM,
4x73GiB scsi disk 2xGigE machine.

\end{abstract}
